# xrwvm_fullstack_developer_capstone

## Repository Name
xrwvm-fullstack_developer_capstone

## Project Name
Fullstack Developer Capstone

## Description
A full-stack Django and React web application that allows users to:
- Browse dealerships
- Filter dealerships by state
- View dealership details and reviews
- Register and log in
- Submit reviews
- View About Us and Contact Us pages

## Tech Stack
- Django
- React
- Bootstrap
- GitHub Actions
- Docker

## Project Structure
- server/frontend/static/About.html
- server/frontend/static/Contact.html
- server/frontend/src/components/Register/Register.jsx
- Django backend APIs for:
  - dealers
  - reviews
  - login/logout
  - car makes/models
  - sentiment analysis
